# Welcome to Telegram UI

Dive into our sleek design on [Figma](https://figma.com/community/file/1348989725141777736/) to get a glimpse of what awaits you.

## Easy Installation

Getting started is a breeze with npm or yarn. Simply run:

**npm:**

```sh
npm install tmaui
```

**yarn:**

```sh
yarn add tmaui
```

## Smooth Development Process

Follow these simple steps to kickstart your development journey:

### 1. Import Styles

Before diving into the development, ensure to import the necessary styles:

```jsx
import 'tmaui/dist/styles.css';
```

### 2. Wrap Your App

Next, wrap your application with `AppRoot` to leverage our platform's features:

```jsx
import { AppRoot } from 'tmaui';

ReactDOM.render(
  <AppRoot>
    <App />
  </AppRoot>,
  document.getElementById('root')
);
```

### Usage example

```jsx
// Import the necessary styles globally
import 'tmaui/dist/styles.css';

// Import components from the library
import { AppRoot, Cell, List, Section } from 'tmaui';

// Example data for rendering list cells
const cellsTexts = ['Chat Settings', 'Data and Storage', 'Devices'];

export const App = () => (
  <AppRoot>
    {/* List component to display a collection of items */}
    <List>
      {/* Section component to group items within the list */}
      <Section
        header="Header for the section"
        footer="Footer for the section"
      >
        {/* Mapping through the cells data to render Cell components */}
        {cellsTexts.map((cellText, index) => (
          <Cell key={index}>{cellText}</Cell>
        ))}
      </Section>
    </List>
  </AppRoot>
);
```

With these steps, you're all set to explore the potential of our library. Happy coding!
